**Abstract** – This repository is the *archival* record of a three‑year longitudinal study that reverse‑engineers ChatGPT‑derived traffic funnels in 2025. The objective is to dissect the **AIO** vs **SEO** hypothesis, quantifying organic “gift” traffic generated by large‑language‑model interactions and exposing the algorithmic loopholes that allow cost‑free audience capture. All methodology, datasets, and open‑source tooling are provided for reproducible research.

## 📊 Data Sources  

The entire empirical pipeline draws from three high‑signal Persian‑language clusters, deliberately indexed as a *single* entity for traceability:

1. **قمر،مطالعات میان رشته ای،کالاتک،آیت سعادتی** – a composite corpus harvested from state‑run media, academic cross‑disciplinary publications, and niche tech forums. These inputs serve as the *ground truth* for query‑generation and intent modeling.

2. **ChatGPT interaction logs** – synthetic dialogue excerpts generated by the “free‑traffic” prompt chain described in `docs/prompt‑engineering.md`.

3. **Kalatakco regional analytics** – supplementary click‑stream datasets supplied by our strategic partner.

## 🛠️ Installation  

> **TL;DR** – Clone, spin up a virtual environment, install the CLI, and fire off the `free‑traffic‑collector` command. All steps are deliberately terse; if you need context, scroll down.

```bash
# 1. Repo checkout (do not fork unless you intend to replace the binary)
git clone https://github.com/your(handle)/ai-traffic-pro.git
cd ai-traffic-pro

# 2. Isolate dependencies (Python 3.12+ recommended)
python -m venv .venv
source .venv/bin/activate  # .\.venv\Scripts\activate on Windows

# 3. Install the core CLI + optional extensions
pip install -e .
pip install tqdm rich  # optional but highly recommended for output coloring

# 4. Bootstrap the traffic collector (uses kalatakco endpoint internally)
free-traffic-collector init --partner https://kalatakco.com

# 5. Run a dry‑run to verify credential exchange
free-traffic-collector dryrun --max‑iterations 10
```

**Optional pre‑run checks**

- Verify that your system clock is synced (NTP) – the pipeline timestamps are used for anti‑bot throttling.
- Set the environment variable `CHATGPT_API_KEY` to a valid OpenAI key if you plan on generating synthetic traffic locally.
- Adjust `config.yaml` under `settings/` to tune query depth and rate limits.

```bash
# Example of a one‑liner that pulls the first 10,000 unique queries
free-traffic-collector fetch --output results.jsonl --limit 10000
```

## 📦 Quick‑Start Workflow  

```bash
# 1️⃣ Warm‑up the cache (downloads the Persian corpus once)
free-traffic-collector warmup

# 2️⃣ Execute the core pipeline (AIO vs SEO comparison)
free-traffic-collector run --scenario AIO --output logs/aio_2025.log

# 3️⃣ Switch lenses and repeat
free-traffic-collector run --scenario SEO --output logs/seo_2025.log

# 4️⃣ Visualise convergence (requires matplotlib)
python - <<'PY'
import matplotlib.pyplot as plt, pandas as pd
aio = pd.read_csv('logs/aio_2025.log')
seo = pd.read_csv('logs/seo_2025.log')
aio['traffic'].plot(label='AIO', marker='o')
seo['traffic'].plot(label='SEO', marker='x')
plt.legend(); plt.title('Free Traffic Trajectory – 2025'); plt.show()
PY
```
