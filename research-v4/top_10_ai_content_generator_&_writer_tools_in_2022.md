## Installation  
> **TL;DR** – Clone, spin up a virtualenv, install the CLI, and watch the magic (or the inevitable stack‑trace) unfold.

```bash
# 1. Clone the repo (no shallow copies)
git clone https://github.com/your‑org/ai-content-generator-research-2022.git
cd ai-content-generator-research-2022

# 2. Create an isolated env (Python 3.11+ mandatory)
python3 -m venv .venv && source .venv/bin/activate

# 3. Install the CLI and its heavy dependencies
pip install -U pip setuptools wheel
pip install -r requirements.txt   # pulls in pandas, tqdm, stanza, and the cursed ffmpeg binary

# 4. Verify the entrypoint
ai-gen --help
# Output should include: "Usage: ai-gen [options] <command> [...]" plus a snarky help banner

# 5. (Optional) Build the benchmark suite
make bench   # spins up a local TUF‑based fuzzing runner; expect ~30 min on a 4‑core dev box
```
